# IBM System i Security
The support for IBM System i security is the same as for other Windows-initiated operations against the mainframe, with the following adjustments:  
  
-   No support for RACF, AFC/2, Kerberos, or Top Secret.  
  
-   Integrate with IBM System i native security system only.  
  
-   Support Single Sign-On via SSO.
