# ISSOLookup2 Interface (COM)

 

The **ISSOLookup2** interface provides a lookup for user credentials associated with an application.

## Requirements

**Type Library:** SSOLookup 1.0 Type Library (SSOLookup.dll)

**Platforms:**  Windows
