# Select Schema Dialog Box

 

Use the **Select Schema** dialog box to select the schema of messaging payload for which you want to create a tracking profile from the list of schemas deployed in the BizTalk Management database (also referred to as the Configuration database).

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Up and Down arrows</strong></td>
<td>Select the schema that contains the payload for which you want to create a tracking profile.</td>
</tr>
</tbody>
</table>


Use the In string text box and Search button to limit the number of activities displayed in the Schema Name window.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>In string text box</strong></td>
<td>Specify part of a schema name to search for in the list of deployed schemas.</td>
</tr>
<tr class="even">
<td><strong>Search button</strong></td>
<td>Start the filter process.</td>
</tr>
</tbody>
</table>
