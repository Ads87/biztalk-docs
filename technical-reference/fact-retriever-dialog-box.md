# Fact Retriever Dialog Box

Â 

Use the **Fact Retriever** dialog box to browse through assemblies to add a reference to a .NET-based fact retriever that provides long-term facts for policy execution.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Browse</strong></td>
<td>Browse through assemblies that implement fact retrievers.</td>
</tr>
<tr class="even">
<td><strong>Assembly</strong></td>
<td>From the drop-down list, select the .NET assembly that contains the currently configured fact retriever, or select <strong>(None)</strong> to clear the configuration.</td>
</tr>
<tr class="odd">
<td><strong>Class</strong></td>
<td>From the drop-down list, select a public .NET class that implements the <strong>IFactRetriever</strong> interface.</td>
</tr>
</tbody>
</table>
