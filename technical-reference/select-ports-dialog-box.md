# Select Ports Dialog Box

Â 

Use the **Select Ports** dialog box to select the port to map to your tracking profile entry.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Up and Down arrows</strong></td>
<td>Select the port that contains the orchestration for which you want to create a tracking profile.</td>
</tr>
<tr class="even">
<td><strong>&gt; button</strong></td>
<td>Move the selected port to the list box on the right, which contains the ports that will be monitored for the data item.</td>
</tr>
<tr class="odd">
<td><strong>&lt; button</strong></td>
<td>Remove the selected ports from the list box on the right.</td>
</tr>
<tr class="even">
<td><strong>&gt;&gt; button</strong></td>
<td>Move all the displayed ports to the list box on the right.</td>
</tr>
<tr class="odd">
<td><strong>&lt;&lt; button</strong></td>
<td>Remove all the ports from the list box on the right.</td>
</tr>
</tbody>
</table>


Use the In string text box and Search button to limit the number of ports displayed in the Port Name window.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Show receive ports check box</strong></td>
<td>Limit the list of displayed ports to receive ports.</td>
</tr>
<tr class="even">
<td><strong>Show send ports check box</strong></td>
<td>Limit the list of displayed ports to send ports.</td>
</tr>
<tr class="odd">
<td><strong>In string text box</strong></td>
<td>Specify part of a port name to search for in the list of configured port names.</td>
</tr>
<tr class="even">
<td><strong>Search button</strong></td>
<td>Start the filter process.</td>
</tr>
</tbody>
</table>
