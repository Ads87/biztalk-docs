# Configuration Wizard, Consolidated Service Accounts View Page

 

Use the **Consolidated Service Accounts View** page to view an aggregated list of features and the accounts used to run the services used in your configuration.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Edit</strong></td>
<td>Edit the service account name of the selected features. <strong>Note:</strong> You can edit multiple features by pressing CTRL and clicking the mouse.</td>
</tr>
</tbody>
</table>
