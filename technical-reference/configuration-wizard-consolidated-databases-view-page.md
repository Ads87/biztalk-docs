# Configuration Wizard, Consolidated Databases View Page

 

Use the **Consolidated Databases View** page to view an aggregated list of features, servers, databases, and data stores used in your configuration.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Edit</strong></td>
<td>Edit the database server name and database name of the selected features. <strong>Note:</strong> You can edit multiple features by pressing CTRL and clicking the mouse.</td>
</tr>
</tbody>
</table>
