# Equal Functoid

Â 

Use the **Equal** functoid ( ![](images/Aa559045.63f9f006-01a4-405d-a197-e9e945144158(BTS.80).jpeg)) to test whether the two input parameters are equal.

## Input

**Parameter 1:** A value to be tested for equality with the parameter 2.

**Parameter 2:** A value to be tested for equality with the parameter 1.

## Output

**Output 1:** The Boolean value **True** if the values of the two input parameters are equal; the Boolean value **False** otherwise.

## Remarks

Input parameter values can be in a variety of data types (string, numeric, or logical). For more information about how the **Logical** functoids handle input parameters of different types, see [Logical Functoids Reference](logical-functoids-reference.md).
