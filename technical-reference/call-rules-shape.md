# Call Rules Shape

 

You can use the **Call Rules** shape to specify a rule policy to be applied using selected orchestration variables as facts.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Configure Policy</strong> property</td>
<td>Specify the policy to be executed by the rule engine. Click the ellipsis (<strong>…</strong>) button to invoke the Policy Configuration dialog box. For more information, see <a href="policy-configuration-dialog-box.md">Policy Configuration Dialog Box</a>.</td>
</tr>
<tr class="even">
<td><strong>Report to Analyst</strong> property</td>
<td>Select <strong>True</strong> if you want to make this shape viewable in the Visual Business Analyst Tool.</td>
</tr>
</tbody>
</table>
