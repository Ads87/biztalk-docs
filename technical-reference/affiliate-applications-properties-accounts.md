# Affiliate Applications Properties: Accounts

 

Use this dialog box to view or modify access accounts for the Affiliate Application.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Application Administrators</strong></td>
<td>The Windows account that will manage this Affiliate Application.</td>
</tr>
<tr class="even">
<td><strong>Application Users</strong></td>
<td>The Windows account that will be used to access this Affiliate Application.</td>
</tr>
</tbody>
</table>
