# Display Name Page

 

Use the **Display Name** page to specify a format for displaying the vocabulary definition in rule conditions and actions.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Parameters</strong></td>
<td>Specify parameters to use, including name, type, value, and zero-based index.</td>
</tr>
<tr class="even">
<td><strong>Edit</strong></td>
<td>Add or edit the value of the selected parameter.</td>
</tr>
<tr class="odd">
<td><strong>Display format string</strong></td>
<td>Type the definition name for display in rule definitions. Use {n} to represent parameters, where n is the zero-based index of the parameter to display.</td>
</tr>
<tr class="even">
<td><strong>Default</strong></td>
<td>Click to reset the display name. The default display string is defined by the definition name plus parameter placeholders.</td>
</tr>
</tbody>
</table>
