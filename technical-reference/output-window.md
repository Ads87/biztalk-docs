# Output Window

 

Use the **Output** window to view results of test execution for a selected policy version.

Use the shortcut menu to access the following options.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Clear All</strong></td>
<td>Clear all text from the Output Window.</td>
</tr>
<tr class="even">
<td><strong>Copy</strong></td>
<td>Copy the selected text in the Output window to the clipboard.</td>
</tr>
<tr class="odd">
<td><strong>Select All</strong></td>
<td>Select all the text contained in the Output window.</td>
</tr>
<tr class="even">
<td><strong>Save to File</strong></td>
<td>Save the text contained in the Output window to a specified file.</td>
</tr>
</tbody>
</table>
