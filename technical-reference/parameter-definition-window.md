# Parameter Definition Window

 

Use the **Parameter Definition** window to specify a parameter value for use in a vocabulary definition display name.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Use constant value</strong></td>
<td>Specify a value to use a constant as the default value for the parameter.</td>
</tr>
<tr class="even">
<td><strong>User definition from published vocabulary</strong></td>
<td>Select a published vocabulary definition of compatible type to use as the default value for the parameter.</td>
</tr>
</tbody>
</table>
