# Configuration Wizard, Business Rules Engine Page

 

Use the **Business Rules Engine** page to configure the Business Rules Engine for this computer.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Enable the Business Rules Engine on this computer</strong></td>
<td>Select <strong>Enable the Business Rules Engine on this computer</strong> to configure this server with the Business Rules Engine.</td>
</tr>
<tr class="even">
<td><strong>Data stores</strong></td>
<td>The Data stores list provides an editable view of the Rules Engine database server name and database name.</td>
</tr>
<tr class="odd">
<td><strong>Windows service</strong></td>
<td>The Windows service list provides and editable view of the account used to run the Rule Engine Update Service.</td>
</tr>
</tbody>
</table>
