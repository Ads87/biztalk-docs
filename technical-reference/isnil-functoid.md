# IsNil Functoid

Â 

Use the **IsNil** functoid (![Logical IsNil functoid](images/Aa561802.427a7da4-80ee-4005-9c42-10b25e143ed6(BTS.80).jpeg "Logical IsNil functoid")) to determine whether the specified input parameter evaluates to **Nil**.

## Input

**Parameter 1:** A value that could be evaluated to **Nil**.

## Output

**Output 1:** The Boolean value **True** if the specified input parameter evaluates to **Nil**; the Boolean value **False** otherwise.
