# ISSOAdmin Interface (COM)

 

The **ISSOAdmin** interface provides administration functions for the Enterprise Single Sign-On (SSO) server database.

## Requirements

**Type Library:** SSOAdmin 1.0 Type Library (SSOAdmin.dll)

**Platforms:**  Windows
