# ISSOConfigSS.RestoreSecret Method

Â 

The **RestoreSecret** method restores master secrets from the password protected backup file.

## Syntax

``` vb
  
Sub RestoreSecret(  
restoreFile as String,  
filePassword as String);  
```

``` csharp
  
void RestoreSecret(  
string restoreFile,  
string filePassword);  
```
