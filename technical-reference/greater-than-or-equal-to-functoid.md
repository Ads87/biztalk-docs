# Greater Than or Equal To Functoid

 

Use the **Greater Than or Equal To** functoid ( ![](images/Aa562017.841a6c54-e118-4ece-ad14-9901823ed172(BTS.80).jpeg)) to test whether the first input parameter is greater than or equal to the second input parameter.

## Input

**Parameter 1:** A value to be tested to determine whether it is greater than or equal to parameter 2.

**Parameter 2:** A value to be tested to determine whether it is greater than or equal to parameter 1.

## Output

**Output 1:** The Boolean value **True** if the value of the first input parameter is greater than or equal to the value of the second input parameter; the Boolean value **False** otherwise.

## Remarks

Input parameter values can be in a variety of data types (string, numeric, or logical). For more information about how the **Logical** functoids handle input parameters of different types, see [Logical Functoids Reference](logical-functoids-reference.md).
