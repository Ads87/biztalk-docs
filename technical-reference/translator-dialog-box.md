# Translator Dialog Box

 

Use the **Translator** dialog box to browse through assemblies to add a reference to a .NET-based translator that translates a policy version into an executable representation for the rule set executor.

<table>
<thead>
<tr class="header">
<th>Use this</th>
<th>To do this</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Browse</strong></td>
<td>Browse through assemblies that implement rule set translators.</td>
</tr>
<tr class="even">
<td><strong>Assembly</strong></td>
<td>From the drop-down list, select the .NET assembly that contains the currently configured translator, or select <strong>(None)</strong> to clear the configuration.</td>
</tr>
<tr class="odd">
<td><strong>Class</strong></td>
<td>From the drop-down list, select a public .NET class that implements the <strong>IRuleSetTranslator</strong> interface.</td>
</tr>
</tbody>
</table>
