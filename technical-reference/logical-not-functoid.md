# Logical NOT Functoid

 

Use the **Logical NOT** functoid (![Logical NOT functoid](images/Aa561635.b4090252-3b23-45c4-ae46-1599420440a4(BTS.80).jpeg "Logical NOT functoid")) to logically negate the value of the Boolean input parameter.

## Input

**Parameter 1:** A value that can be evaluated as either **True** or **False**.

## Output

**Output 1:** The Boolean value **True** if the specified input parameter evaluates to **False**; the Boolean value **False** otherwise.

## Remarks

The input parameter value can be in a variety of data types (string, numeric, or logical), although passing a string to this functoid does not yield a meaningful result. For more information about how the **Logical** functoids handle input parameters of different types, see [Logical Functoids Reference](logical-functoids-reference.md).
